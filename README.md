# Thejaswini & Chandrasekar — Wedding Showcase

A responsive, single-page wedding website built with plain HTML, CSS and JavaScript.

## Add the remaining photographs

The supplied photograph is already installed as `assets/images/hero.jpg`.

Replace the nine SVG placeholders in `assets/images/` with your own JPG photographs using these exact names:

- `photo-02.jpg`
- `photo-03.jpg`
- `photo-04.jpg`
- `photo-05.jpg`
- `photo-06.jpg`
- `photo-07.jpg`
- `photo-08.jpg`
- `photo-09.jpg`
- `photo-10.jpg`

Then update each corresponding path in `index.html` from `.svg` to `.jpg`.

For best results, export photographs at approximately 1600–2400 pixels on the long edge and keep each file below about 1.5 MB.

## Publish with GitHub Pages

1. Sign in to GitHub and select **New repository**.
2. Name it, for example, `thejaswini-chandrasekar-wedding`.
3. Set visibility to **Public** and create the repository.
4. Select **Add file → Upload files**.
5. Drag the complete contents of this folder into the upload area. Preserve the `assets/images` folder structure.
6. Enter a commit message such as `Publish wedding website` and select **Commit changes**.
7. Open **Settings → Pages**.
8. Under **Build and deployment**, select **Deploy from a branch**.
9. Choose branch **main**, folder **/(root)**, then select **Save**.
10. Wait one or two minutes. GitHub will display the website URL in the Pages section.

The usual URL format is:

`https://YOUR-GITHUB-USERNAME.github.io/thejaswini-chandrasekar-wedding/`

## Local preview

Open `index.html` directly in a browser, or use VS Code with the Live Server extension.
